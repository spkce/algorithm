### 102. 二叉树的层序遍历 - 力扣（LeetCode）

给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

示例 1：
```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```
示例 2：
```
输入：root = [1]
输出：[[1]]
```
示例 3：
```
输入：root = []
输出：[]
```

提示：
```
树中节点数目在范围 [0, 2000] 内
-1000 <= Node.val <= 1000
```

```c++
class Solution {
public:
	vector<vector<int>> levelOrder(TreeNode* root) {
		std::vector<std::vector<int>> ans;
		if (!root)
		{
			return ans;
		}
		std::queue<TreeNode*> que;
		que.push(root);

		while (!que.empty())
		{
			std::vector<int> t;
			for (int i = que.size(); i > 0; i--)
			{
				TreeNode* p = que.front(); 
				t.push_back(p->val);
				que.pop();
				if (p->left != nullptr)
				{
					que.push(p->left);
				}

				if (p->right != nullptr)
				{
					que.push(p->right);
				}
			}
			ans.push_back(t);
		}
		return ans;
	}
};
```
